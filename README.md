# Browsly Downloads

Official release repository for **Browsly**.

This repository hosts Windows installers and release assets for Browsly. It is used by the application's auto-update system and provides a central place to download the latest stable releases.

> **Note**
> This repository only contains release binaries. The application source code is available in the main Browsly repository.

---

## 📥 Download

Download the latest version from the Releases page:

👉 https://github.com/browsly/browsly-downloads/releases/latest

Latest release assets may include:

- Windows Installer (.exe)
- Portable Version (.zip)
- Release Notes
- Checksums (if available)

## 🚀 About Browsly

Browsly is an offline-first visual automation platform that transforms Node.js libraries into drag-and-drop workflow nodes.

Features include:

- 🎯 Visual Workflow Builder
- 📦 Library Marketplace
- 🔌 Node.js Package Integration
- ⚡ Local Workflow Execution
- 🖥 Offline Support
- 🧠 AI-powered Workflow Assistance
- 📅 Workflow Scheduler
- 🌐 Local API Endpoints
- 📊 Execution Logs & Monitoring
- 🔄 Auto Updates
